
Cool Message module.
--------------------

Provides a beautiful display of system messages (errors, warning, etc.).
These messages displaying at the top of the page using CSS3 and jQuery.
If you click on message then it will hide.


BASED ON
--------

Cool notification messages with CSS3 & jQuery
by Catalin Rosu, a.k.a Red.
http://www.red-team-design.com/cool-notification-messages-with-css3-jquery.


INSTALL
-------

Just download, unzip/untar and activate the module.
(More info about installing contributed modules:
  http://drupal.org/documentation/install/modules-themes/modules-7)


Home page: http://drupal.org/project/cool_message
